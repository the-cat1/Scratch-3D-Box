# Scratch-3D-Box
**一个使用Scratch3制作的3D游戏(可能是吧😁)。**
~~抄~~参考了一个[翻译的教程](https://www.bilibili.com/video/BV1eK411J7n5)。

## 如何运行它
使用Scratch3.0打开 ```Scratch3D.sb3```，点击小绿旗启动游戏。

## 如何玩它
按下以下按键就有对应功能。
- Q 下降
- E 上升
- W 前进
- S 后退
- A 左走
- D 右走
- <上方向键> 向上看
- <下方向键> 向下看
- <左方向键> 向左看
- <右方向键> 向右看

游戏中有一个立体的正方体，可以根据视角变成一个立体形状。
